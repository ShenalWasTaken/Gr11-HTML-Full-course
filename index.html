HTML Hyperlinks

  • hyperlinks are used to link a URL to a text in HTML
  • The letter 'a' use to the opening tag and the closing tag of a hyper link
  • 'href' attribute is used in the opening tag to define the website
  • between the opening and the closing 'a' tags we can put a text or an image to link an URL
  • the target attribute is used in the opening 'a' tag to define wether if the link needed to open in a new tab or in the same tab
  • the default target attribute value is '_self' which will open the link in the same tab but changing the value from '_self' to '_blank' will open the link in a new tab

Table

  • When using/inserting table to an HTML there are 5 tags needed to know about

    1. table- this is the opening tag and the closing tag of a table
    2. tr-this tag is use to open a new row in the table
    3. th-this is for the tabe header,By default, table headings are bold and centered. th tag is used between the closing and opening tag in tr tag
    4. td- this tag is like the th tag but not for the header but for the cells
    5. caption-to put a heading to the table on a desired location

  • there are also lots attribute when using table in HTML

    1. align-when used in the table tag its shows how the should be align in the page ,left, center, right are the values

    2. align inside the caption tag - use to define the location of the caption that used in the table, left, right,top, bottom, center are the values

    3. width-define the width of the tabel or any cell with pixel, rem.

    4. heigh-define the lenth of the tabel or any cell with pixel, rem.

    5. bgcolor-define the background color of the table

    6. border- the values are '1' and '0' to tell if a border is needed to the table or not

    7. border color-define the border color in name or hex values

    8. background-can be used to insert an image as the background in the table

    9. cellpadding-define the spacing between the cellborder and the cell content

    10. celllspacing-define the spacing between cells

    11. colspan-used in the td tage to connect multiple cells in the row , the value is set as how many cells needed to be conected
